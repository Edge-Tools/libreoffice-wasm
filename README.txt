libreoffice-wasm — Corresponding Source mirror
==================================================

This repository publishes the Corresponding Source for the WebAssembly
build of libreoffice (license: MPL-2.0) served from cdn.edgetools.io.

Contents
  build/      our build recipe: Dockerfile + helper scripts/config/patches.
              Rebuild with:  docker build build/
  upstream    too large for a git tree — published as Release assets
              under the "source" tag (download links below).
